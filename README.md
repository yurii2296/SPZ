Розробка програмного забезпечення для керування базами даних. Розробка програми, що дозволяє керувати базами даних та виконувати різні запити до них

• Створення таблиць з можливістю задання власних назв стовпців
• Ручне додавання нових записів у таблиці бази даних
• Автоматична генерація тестових даних для перевірки роботи алгоритмів
• Видалення всіх даних з обраної таблиці
• Сортування інформації за числовими показниками (наприклад, за балами)
• Візуалізація та виведення вмісту бази даних у консоль
• Робота з файлами баз даних формату .db
• Коректне відкриття та закриття з’єднання з базою

Розробка мовою програмування C++ з використанням бібліотеки SQLite3 та стандартних бібліотек вводу/виводу.
